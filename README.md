Сайт позволяет рассчитать цену скина с наклейками.
1) Выбери скин из списка
2) Выбери наклейки, расположенные на скине
3) Получи прайс 

Цена рассчитывается по формуле:<p>
***P(full) = P(skin) + P(stickers)/10***<p>
! Данная формула не является гарантией.
Какие-то комбинации могут стоить дороже или дешевле.
