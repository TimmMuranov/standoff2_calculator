## Сайт позволяет рассчитать цену скина с наклейками
1) Скачай данный репозиторий.<p>
2) Открой index.html в браузере.<p>
3) укажи прайс голого скина и добавь на него наклейки кнопками "+" и "-".<p>
4) Нажми "Рассчитать" и получи реальный прайс скина :)<p>

### Добавить наклейки можо так:
1) Добавить фото наклейки в папку "stickers" (можно добавлять папки с картинками)<p>
2) В файле "price_stickers.js" добавить коллекцию и прайсы за наклейки в формате:
```
collection_name : {
    "stiker1_name" : 30.00,
    "sticker2_name" : 5000
},
```
*3) чтобы отображались картинки стикеров, добавьте в папку "stickers" папку с именем коллекции и в эту папку добавьте фото наклеек с именами в точности как из файла и расширением jpg.<p>
