## Сайт позволяет рассчитать цену скина с наклейками
1) Скачай данный репозиторий.<p>
2) Открой index.html в браузере.<p>
3) нажми "Выбрать папку" и выбери папку stickers из данного репозитория.<p>
4) укажи прайс голого скина и добавь на него наклейки кнопками "+" и "-".<p>
5) Нажми "Рассчитать" и получи реальный прайс скина :)<p>

### Добавить наклейки можо так:
1) Добавить фото наклейки в папку stickers (можно добавлять папки с картинками)<p>
2) В файле "price_stickers.js" добавить в фигурные скобки запись вида:<p>
```"sticker_name" : 20.00, //укажите актуальный прайс```
3) Перезагрузить страницу и заново добавить папку stickers.<p>

Можно добавлять только определенные коллекции. Для этого выберите нужную папку в stickers.<p>
